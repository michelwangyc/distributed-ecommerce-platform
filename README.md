# Server

The server is a Spring Boot application deployed to AWS ECS.

POST endpoint:

POST /products

Returns:
HTTP 201 Created

The server does not persist data. It immediately returns 201 after receiving a valid request.

To run locally:

mvn spring-boot:run

To deploy:
Build Docker image
Push to ECR
Run in ECS

# Client Part 1 – Multithreaded Load Generator

Purpose:
Send 200,000 POST requests using multiple threads.

Features:
- Fixed thread pool
- Progress printed every 1000 requests
- Wall time measurement
- Throughput calculation

To run:

cd client-part1
mvn clean package
java -cp target/classes edu.neu.cs6650.ClientPart1Main

Arguments (optional):

ClientPart1Main <baseUrl> <threads> <totalRequests>

Example:

java -cp target/classes edu.neu.cs6650.ClientPart1Main http://<ECS-IP>:8080 8 200000

Output:
- Threads
- Total requests

# Client Part 2 – Instrumented Client

Purpose:
Add detailed latency recording and performance statistics.

Additional Features:
- Dedicated product generator thread
- BlockingQueue for products
- BlockingQueue for logging
- CSV writer thread
- Retry up to 5 times for 4XX/5XX
- Latency recording per request
- Statistics calculated after completion:
  - Mean
  - Median
  - P99
  - Min
  - Max
- Instantaneous throughput over time

To run:

cd client-part2
mvn clean package
java -cp target/classes edu.neu.cs6650.ClientPart2Main

Arguments (optional):

ClientPart2Main <threads> <totalRequests> <baseUrl>

Example:

java -cp target/classes edu.neu.cs6650.ClientPart2Main 8 200000 http://<ECS-IP>:8080

Output files:
- client2.csv
- throughput_over_time.csv

# Design Notes

- Each worker thread creates its own ApiClient instance.
- Products are generated by a single producer thread.
- Workers consume products from a BlockingQueue.
- CSV writing is offloaded to a separate thread to avoid slowing down uploads.
- 201 is returned instead of 200 because POST creates a new resource.
- Expected multi-threaded throughput is approximately:
  
  threads × single-thread throughput

Performance scaling should be close to linear.
